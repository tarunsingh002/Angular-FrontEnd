  • Website that allows users to sign up and login to browse and/or buy from a catalogue of electronic products
  • Search bar on the home page allows the user to search products by name, brand or category.
  • Customer(User) facing side of the website includes:
    • Facility to add products to the cart via the home page or the product details page.
    • The cart page to edit the quantity of the individual products or remove them from the cart
    • Checkout page to make a payment and purchase the products on the cart.
    • Orders page to view all the previous orders placed by the user
    • Wishlist page to view, add or remove products to their personal Wishlist.
  • Admin facing side of the website includes:
    • Facility to add/delete/update any product.
    • All Orders page that displays a clickable list of all the orders placed on the website and total earnings.
    • Order Details page that lists all of the products and their quantity in the order.
    • Wishlist Data for all Products that lists all of the Products and the number of times it has been wishlisted.
  • Used Bootstrap 4 as CSS framework for styling. Deployed Github repo for Front End Angular SPA to Vercel.
  • Used Spring Boot 3 to build the REST API and deployed it to Azure.
  • Used Azure SQL Database as database for the application.
